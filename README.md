# Financial-Modeling-Python

## English Version
This project, developed during my MIASHS Bachelor's degree at the University of Nantes, is a Python-based tool designed to simulate complex loan repayment schedules.

**Key Features:**
* Support for multiple repayment types: Constant Annuities, Constant Amortization, In Fine, and Geometric Progression.
* Grace period management (deferred payments).
* Built with: Python, Pandas, NumPy.

---

## Version Française
Ce projet, réalisé en Licence MIASHS à l'Université de Nantes, est un outil de modélisation financière permettant de générer des tableaux d'amortissement complexes.

**Points clés :**
* Gestion de plusieurs types de remboursements : Annuités constantes, Amortissements constants, In Fine et Progression géométrique.
* Gestion des périodes de différé.
* Technologies : Python, Pandas, NumPy.
